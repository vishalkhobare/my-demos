Chess

This game allowed to play chess either on same machine or online.

  1. Play chess on same machine
    1. Go to here
    2. Choose option Two Player
    3. Start playing chess alternatively.
    4. Note: This type of game that both users on same machine
  2. Play chess online
    1. Go to here
    2. Choose option Online. Then enter your name and click on "Submit My name". You will now be added to online users' list.
    3. As no user is enrolled so far except you, below message will appear.
    4. Now, open same URL in another browser and follow steps from 1 to 3. You should be able to see the first user enrolled in online users' list. Check the check box in front of the online user and press Connect button. Now, the game request will be sent to the other user.
    5. Go to second browser, where you should be able to see the game request sent by first user. Click on Yes link to start the game.
    6. The user with black pawns will have to wait for his turn.
    7. Meanwhile, user with white pawn can make first move.
